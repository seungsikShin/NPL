# 실적 관리 AI 시스템

월별 실적 관리 및 AI 분석을 제공하는 웹 애플리케이션

## 기능
- 사용자 인증 (로그인/회원가입)
- 실적 데이터 시각화 (Radar Chart)
- AI 기반 맞춤 제안
- 관리자 대시보드
- 실시간 데이터 필터링

## 기술 스택
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Firebase Authentication & Realtime Database
- AI Integration: Make.com + GPT API
- Charts: SVG + Custom Animations

## 설치 및 실행
1. 저장소 클론
2. Firebase 설정 파일 구성
3. index.html 실행

## 폴더 구조
- `css/` - 스타일시트
- `js/` - JavaScript 모듈들
- `docs/` - 문서 및 가이드# NPL
